DivBase is a service built and maintained by SciLifeLab Data Centre that enables life science researchers at Swedish institutions and their collaborators to manage, explore, and query genomic variants in VCF format alongside associated sample metadata. The service provides a secure platform for managing genomic variants and metadata files for non-human and non-sensitive data.

Key features

Share

Import and export VCF and metadata files

Secure

Secure user authentication and authorisation

Version

File versioning and rollback

Organise

Files organised into projects

Collaborate

Multiple user roles and permission levels

Explore

Query your data and filter and merge datasets

DivBase is under development

DivBase is currently in pre-release stage, which means that it is still under active development and testing. At this stage we have only opened up the platform to a limited number of users for testing and feedback collection. If you are interested in trying out DivBase and providing feedback, please contact us.
